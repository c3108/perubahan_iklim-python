# Website Edukasi tentang Deforestasi dan Perubahan Iklim
Website ini merupakan platform edukatif interaktif yang bertujuan untuk meningkatkan kesadaran masyarakat, khususnya pelajar, tentang deforestasi, dampaknya terhadap lingkungan, serta pentingnya pelestarian hutan. Fitur-fitur yang disajikan dalam website ini dibuat agar menarik dan mudah dipahami.

Website ini mencakup:
- Informasi
Penjelasan tentang deforestasi
Penyebab utama dan dampaknya

- Kuis Edukatif
Pertanyaan berbasis materi pada information page
Membantu pengguna menguji pemahaman mereka tentang deforestasi

- Merekam pohon yang ditanam
Counter sederhana untuk mensimulasikan penanaman pohon
Setiap interaksi pengguna akan menambah jumlah pohon yang ditanam
Counter mencatat total pohon yang berhasil ditanam oleh pengguna

Secara keseluruhan, website ini bertujuan menanamkan rasa tanggung jawab terhadap lingkungan kepada pengguna.
> Dibuat dengan Python, HTML, CSS
